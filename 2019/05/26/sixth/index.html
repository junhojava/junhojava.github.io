Extjs-CE Quick Start

Sencha에서 무료로 사용할 수 있는 CE를 내놓았다. Extjs를 아직 구입하지 않은 사용자에게 좋은 시작이 될 수 있다 느껴졌다.
Extjs- CE를 사용하기 위해서는 약간의 요구사항을 만족하는데 그것을 나열해 보겠다.

  1. Node 10+
  2. NPM 6+
  3. JAVA 8 && 11

이 요구사항이 만족되면 https://www.sencha.com/products/extjs/communityedition/ 페이지로 이동하여 자신의 정보를 입력한다.
